# 108. 将有序数组转换为二叉搜索树

## 题目

给你一个整数数组` nums` ，其中元素已经按 **升序** 排列，请你将其转换为一棵 **高度平衡** 二叉搜索树。

**高度平衡** 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

**示例1：**
> 输入：nums = [-10,-3,0,5,9]
> 输出：[0,-3,9,-10,null,5]
> 解释：[0,-10,5,null,-3,null,9] 也将被视为正确答案

**示例2：**

> 输入：nums = [1,3]
> 输出：[3,1]
> 解释：[1,3] 和 [3,1] 都是高度平衡二叉搜索树。

## 解题思路

递归。

时间复杂度为O(n)，空间复杂度为O(logn)。

## 提示

* `1 <= nums.length <= 10^4`
* `-10^4 <= nums[i] <= 10^4`
* `nums` 按 **严格递增** 顺序排列
