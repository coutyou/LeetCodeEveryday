# 94. 二叉树的中序遍历

## 题目

给定一个二叉树的根节点 `root` ，返回它的 **中序** 遍历。

## 示例

**示例 1：**

>       输入：root = [1,null,2,3]
>       输出：[1,3,2]

**示例 2：**

>       输入：root = []
>       输出：[]

**示例 3：**

> 输入：root = [1]
> 输出：[1]

**示例 4：**

> 输入：root = [1,2]
> 输出：[2,1]

**示例 5：**

> 输入：root = [1,null,2]
> 输出：[1,2]

## 提示

* 树中节点数目在范围 `[0, 100]` 内
* `-100 <= Node.val <= 100`

## 进阶

递归算法很简单，你可以通过迭代算法完成吗？

## 解题思路

递归的方法很简单，迭代的方法，我们通过观察可以知道，对于每个节点，不断向左延伸，直到没有左子树，然后把当前节点的值加入结果，然后切换到右子树，再进行上述过程。处理完当前节点的左右子树后，进行回溯，因此可以用一个栈来记录向左延伸时经过的节点。

时间复杂度为O(n)，空间复杂度为O(n)。
