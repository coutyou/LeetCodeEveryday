# 653. 两数之和 IV - 输入 BST

## 题目

给定一个二叉搜索树 `root` 和一个目标结果 `k`，如果 BST 中存在两个元素且它们的和等于给定的目标结果，则返回 `true`。

**示例1：**

>    输入: root = [5,3,6,2,4,null,7], k = 9
>    输出: true

**示例2：**

> 输入: root = [5,3,6,2,4,null,7], k = 28
> 输出: false

**示例3：**

> 输入: root = [2,1,3], k = 4
> 输出: true

**示例4：**

> 输入: root = [2,1,3], k = 1
> 输出: false

**示例5：**

> 输入: root = [2,1,3], k = 3
> 输出: true

## 提示

* 二叉树的节点个数的范围是 `[1, 10^4]`
* `-10^4 <= Node.val <= 10^4`
* `root` 为二叉搜索树
* `-10^5 <= k <= 10^5`

## 解题思路

遍历二叉树得到所有节点的哈希表，然后遍历该哈希表找另一个数是否在表中。因为二叉搜索树没有重复的元素，因此还要确定另一个数不等于本身。

时间复杂度：O(n)。

空间复杂度：O(n) 。
